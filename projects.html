ACTINIUM - Shell C++ Modulaire

C++ Windows Shell DLL

Description

ACTINIUM est un shell C++ qui permet de modifier le comportement des commandes à la volée. Contrairement aux shells traditionnels, ACTINIUM donne un accès direct au code source de chaque commande depuis le terminal, permettant de les modifier et de les recompiler sans quitter l'environnement.

Fonctionnalités

  • Interface en ligne de commande avec support UTF-8
  • Navigation dans le système de fichiers (ls, cd, pwd)
  • Exécution de commandes externes via variables d'environnement
  • Modification et recompilation des commandes à la volée
  • Architecture modulaire pour faciliter l'ajout de nouvelles commandes

Architecture technique

  • Développé en C++ (standard C++11)
  • Architecture modulaire basée sur les DLL
  • Compilation intégrée via appels système
  • Gestion dynamique de la mémoire et des ressources système

Touch on Windows

C++ Windows API Utilitaire

Description

Implémentation de la commande Unix "touch" pour Windows, permettant de créer des fichiers vides ou de mettre à jour la date de modification d'un fichier existant.

Fonctionnalités

  • Création de fichiers vides
  • Mise à jour de l'horodatage des fichiers existants
  • Option pour ouvrir automatiquement le fichier après sa création
  • Interface en ligne de commande simple

Architecture technique

  • Développé en C++ avec l'API Windows
  • Utilisation de CreateFile et SetFileTime pour la gestion des horodatages
  • ShellExecute pour l'ouverture des fichiers
