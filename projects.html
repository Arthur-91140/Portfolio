Emploi du Temps IUT d'Orsay

HTML/CSS JavaScript API REST Responsive Moderne

Description

Application web moderne et intuitive permettant aux étudiants de l'IUT d'Orsay de consulter leur emploi du temps avec une interface claire et responsive. Le projet vise à simplifier l'accès aux informations académiques tout en offrant une expérience utilisateur optimisée.

Fonctionnalités Principales

  • Authentification simplifiée via identifiant au format prénom.nom
  • Vue journalière avec informations détaillées sur les cours (horaires, salles, enseignants)
  • Vue hebdomadaire pour visualiser l'emploi du temps complet
  • Navigation temporelle fluide entre les jours et semaines
  • Filtrage par type de cours (TP, CM, TD, Projet, etc.)
  • Design responsive adapté à tous les appareils (mobile, tablette, desktop)
  • Interface moderne avec animations et micro-interactions
  • Mode sombre/clair automatique selon les préférences système

Architecture Technique

  • Frontend : HTML5 sémantique, CSS3 moderne avec variables et flexbox/grid
  • JavaScript : ES6+ vanilla avec gestion d'état côté client
  • API : Intégration avec l'API officielle de l'IUT d'Orsay
  • Performance : Optimisations pour un chargement rapide et fluide
  • Accessibilité : Conformité WCAG 2.1 pour tous les utilisateurs

Défis Techniques Relevés

  • Gestion des données temporelles et des fuseaux horaires
  • Optimisation des performances pour le rendu de calendriers complexes
  • Adaptation responsive pour différentes tailles d'écran
  • Intégration fluide avec l'API existante de l'université

ACTINIUM - Shell C++ Modulaire

C++17 Windows API Modulaire DLL Innovation

Concept Innovant

ACTINIUM révolutionne l'interaction avec les shells traditionnels en permettant de modifier et recompiler les commandes directement depuis le terminal. Cette approche unique offre une flexibilité inégalée pour l'expérimentation et le développement rapide.

Fonctionnalités Révolutionnaires

  • Modification en temps réel du code source des commandes
  • Compilation automatique et rechargement à chaud des modules
  • Interface en ligne de commande avec support UTF-8 complet
  • Navigation avancée dans le système de fichiers (ls, cd, pwd améliorés)
  • Exécution transparente des commandes système existantes
  • Système de plugins extensible pour nouvelles fonctionnalités
  • Historique intelligent des commandes avec auto-complétion
  • Gestion avancée des erreurs avec suggestions de correction

Architecture Avancée

  • Langage : C++17 moderne avec STL optimisée
  • Modules : Architecture DLL pour maximum de flexibilité
  • Compilation : Intégration native avec les compilateurs système
  • Mémoire : Gestion intelligente avec RAII et smart pointers
  • Performance : Optimisations low-level pour vitesse maximale
  • Portabilité : Couche d'abstraction pour multi-plateforme

Innovations Techniques

  • Premier shell permettant la modification de code en live
  • Système de compilation incrémentale intégré
  • Architecture plugin révolutionnaire pour l'extensibilité
  • Gestion avancée des dépendances entre modules

Touch on Windows

C++ Windows API Utilitaire Timestamps Léger

Description

Port élégant et efficace de la commande Unix "touch" pour l'environnement Windows. Cet utilitaire compact comble une lacune importante en apportant cette fonctionnalité essentielle aux développeurs Windows.

Fonctionnalités Complètes

  • Création instantanée de fichiers vides avec gestion d'erreurs
  • Mise à jour précise des horodatages (accès, modification, création)
  • Option d'ouverture automatique avec l'éditeur par défaut
  • Support des chemins relatifs et absolus
  • Gestion des caractères spéciaux et espaces dans les noms
  • Interface en ligne de commande intuitive et documentée
  • Mode verbose pour debugging et apprentissage

Excellence Technique

  • Performance : Optimisé pour vitesse et faible empreinte mémoire
  • Compatibilité : Support de Windows 7 à Windows 11
  • API Native : Utilisation directe de CreateFile et SetFileTime
  • Robustesse : Gestion complète des cas d'erreur et validation
  • Simplicité : Interface épurée fidèle à l'original Unix

Impact & Utilisation

  • Facilite la transition des développeurs Unix/Linux vers Windows
  • Améliore les workflows de développement cross-platform
  • Utilisé dans des scripts d'automatisation et de build
  • Contribue à l'écosystème open source Windows
